Czerny Op. 599 — Module Guide

How It Works

Three etudes from Czerny's Op. 599. All exercises stay in C position (C–D–E–F–G) — no position shifts needed. Each etude trains a different coordination skill between the hands. Play each note as it lights up on the staff.

Etudes
No. 1RH plays quarter notes (C–D–E–F–G–F–E–D), LH holds a whole note C. Trains right hand independence.
No. 2Both hands play quarter notes in parallel motion up and down. Trains hand coordination.
No. 3RH plays eighth notes while LH plays quarter notes. The hardest coordination challenge of the three.
No. 4Full C major scale in both hands with thumb crossing. Key skill for smooth scale playing.
No. 7Hands alternate on every note. Trains evenness and independence of touch.
HintsHints cycles through: 1× note names → 2× + fingering → 3× + keyboard LED guide → 4× all off.
Learning Tip

Start with No. 1 to get comfortable with the C position. Then move to No. 2 for both hands together. Only attempt No. 3 once No. 2 feels natural — the rhythm difference between hands is tricky at first.
